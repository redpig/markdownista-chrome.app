Markdownista Chrome App
=======================

Markdownista is a application which can concentrate to typing the sentences or articles using Markdown notation.

[Chrome Webstore](https://chrome.google.com/webstore/detail/kbaamlaoekkcaebldgfmoogffhjhdpha)


FEATURES
--------

- HTML Preview
- Save to local storage
- Load from local storage


KEYBOARD SHORTCUT
-----------------

- CTRL+ENTER で簡単にプレビュー＆戻る
- CTRL+S でパッと保存
- CTRL+L でサッと読み込み
